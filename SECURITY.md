# Security Policy

- Report vulnerabilities via GitHub Security Advisory or email: adzextreem@gmail.com.
- Please do not open public issues for security reports.
- We aim to acknowledge within 72 hours and provide a fix ETA within 7 days for high severity issues.

## Supported Versions
- Active development branch: `main`

## Scope
- This repository and officially published packages/artifacts.

## Responsible Disclosure
- We follow coordinated disclosure best practices. Thank you for helping keep users safe.
